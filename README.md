# Silicon Valley Software House

Landing page for Silicon Valley Software House — a team of Silicon Valley engineers returning to Thailand.

## Development

Open `index.html` in a browser, or use a local server:

```bash
npx live-server
```

## Deployment

Static site — push to `main` and enable GitHub Pages (deploy from root).

## Waitlist Form Setup

1. Create a Google Form with one "Email" field
2. Get the form action URL and entry field name
3. Update `GOOGLE_FORM_URL` and `GOOGLE_FORM_EMAIL_FIELD` in `js/main.js`
